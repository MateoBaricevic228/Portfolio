O Meni

Front End Web Developer

Zovem se Mateo i imam 26 godina. Trenutno živim u Zadru. Završio sam srednju strukovnu školu, smjer elektroinstalater, ali u tome se nisam pronašao. Oduvijek me zanimalo kako nastaju web stranice. Općenito rad na računalu mi je odgovarao i u tome sam se lagano pronalazio. Sve je krenulo od prvog saznanja o programiranju i baviti se takvim poslom. Tu sam počeo istraživati i probudila se zainteresiranost. Kad sam saznao kao se dijele programeri i kako koji radi, vidio sam da imam ideju izgleda same stranice pa sam pratio i rad web dizajnera. Htio sam stvarati i napravit nešto što se vidi, počeo sam učiti HTML, CSS uz to i SASS pa JavuScript. Tu sam se odlučio za poziciju frontend developera. Bio sam sretan kod prvog stvaranja svoje web stranice, odmah od početka sam učio da ima mobilna, tablet i desktop verzija. Osoba sam koja se želi i dalje usavršavati i učiti, pa sam uz to počeo i na Wordpress-u slagati stranice što mi je bilo jako zanimljivo. Inače volim glazbu sviram, pjevam, uvijek dobro raspoložen, pristupačan, odgovoran, i spreman na razne izazove.
